Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the JS DOM
1. document.getElementById(),
2. document.getElementsByClassName(),
3. document.getElementsByTagName(),
4. document.querySelector(),
5. document.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

* innerHTML returns the HTML content like tags.
* innerText returns only the visible text in a HTML element.
* And the textContent returns the all text include the hidden text inside in element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we can add a single eventListeners to the parent element and we can handle it on the child elements by using event.target property.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept in the JS DOM where event starts from the targeted element and bubbles up to its parent,grandparent and root element unless it's stopped.

Question-5: How do you create, add, and remove elements using JavaScript?
1. To create elements in JS i use document.createElement()
2. To add elements in JS i use .add() or add something in last div we can use appendChild()
3. To remove elements in JS i use .remove() or .removeChild()
